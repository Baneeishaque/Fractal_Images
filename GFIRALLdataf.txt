GFIRALLdataf.txt
================

Data tables are from:
 * https://en.wikipedia.org/wiki/Barnsley_fern (Barnsley and Thelypteridaceae
   Fern Fractals)
 * http://cs.lmu.edu/~ray/notes/ifs/
 * http://paulbourke.net/fractals/
 * http://ecademy.agnesscott.edu/~lriddle/ifskit/gallery/gallery.htm

Find all samples of dat-files in GPRFDATA folder.  

Remarks:
  * All tables presented in data files for IFS fractals listed below.
  * Only for prime and selected IFS fractals testing samples.
  * Put such files in the R working directory, e.g., in "RData" on C drive.
  * NOTE: These are Windows dat-files.

NOTES:
http://cs.lmu.edu/~ray/notes/ifs/

By convention an IFS table is written in n rows and 7 columns of numbers in the form

 a  b c  d    e      f      p
0.5 0 0 0.5  0.00  0.000  0.3333
0.5 0 0 0.5  0.50  0.000  0.3333
0.5 0 0 0.5  0.25  0.433  0.3334

But all that you need to know is following:
 1. Using Name/Table below create text dat-file with suggested name (or any name you want, 
    but matching name from the sample in GFIRALLsamples.R).
 2. Put n lines (followed below the name here) on the top of created text dat-file.
 3. Use related sample from GFIRALLsamples.R (or create new one) to plot
    this fractal.

That's it!
